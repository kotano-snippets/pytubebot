# Pytube bot

Telegram bot for downloading video from youtube.

## Requirements

[Dependencies](./requirements.txt)

Requires `config.json` file inside project directory:

```json
{
    "token": "BOTTOKEN",
    "test_token": "TESTTOKEN",
    "test_chat_id": "CHATID"
}
```

## Usage

1. Activate bot.
2. Send one or more youtube video links.
3. Get your video.
